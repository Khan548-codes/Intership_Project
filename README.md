# Task Manager (Student Project)

This repository is for my internship project: **Task Manager Web App**.

## Goal
Create a simple web-based task manager where a user can:
- Add tasks
- Mark tasks as complete
- Delete tasks
- Save tasks in the browser (LocalStorage)

## Tools (planned)
- HTML
- CSS
- JavaScript
- LocalStorage

## Work Plan (2 weeks)
Week 1:
- Create the basic UI (input + button + task list)
- Add tasks using JavaScript
- Show tasks on the screen

Week 2:
- Mark task complete
- Delete task
- Save/load tasks using LocalStorage
- Final checks and small improvements
